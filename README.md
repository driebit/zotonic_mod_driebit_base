zotonic_mod_driebit_base
========================

A Zotonic module that adds legacy Ginger templates, search and other features.

This module copies parts of [Ginger](http://github.com/driebit/ginger).

This is meant as a starter for a generic ginger base module to incrementally add to
whatever new upgrade candidate sites need.

Contents
--------

- templates
- javascript
- dispatch rules

  - page
